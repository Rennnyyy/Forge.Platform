# SLICING — Forge.Entity.Tests.Fixtures.Core

Lightweight fixture layer per [root ADR-0007](../../adr/0007-shared-test-fixtures.md)
and [root ADR-0010](../../adr/0010-slice-folder-structure.md).

## Purpose

Provides the canonical sample entity types and `EntityOptionsFixture` to test projects
that exercise `IEntityStore` / `ISparqlQueryStore` directly and do not need the
active-record surface (`.CreateAsync()`, `.ListAsync()`, etc.) generated by
`Forge.Operations.Generators`.

Test projects that need active-record methods reference the full
`Forge.Entity.Tests.Fixtures` project, which adds `Forge.Operations` and
`Forge.Operations.Generators` on top of the same conceptual domain model.

## Design constraint — source-generator partial classes

C# partial classes can only be extended within the same assembly. Because
`Forge.Operations.Generators` generates `partial class Artist { void CreateAsync() … }`
in the compiling assembly, the entity *type definitions* must reside in the same project
that references the generator. This means:

- `Entity.Tests.Fixtures.Core` defines entity types **without** Operations — the
  generator does not run, so no active-record methods are emitted.
- `Entity.Tests.Fixtures` defines the **same** entity types **with** Operations — the
  generator runs and emits `.CreateAsync()` etc. on each type.

The entity type definitions in `Sample/` are intentionally identical across both projects.
When modifying a sample entity class, update both copies in the same commit.

## Dependencies

| Package / Project | Reason |
|-------------------|--------|
| `Forge.Entity` | Entity base types, `IEntity`, `EntityOptions`. |
| `Forge.Entity.Generators` | Analyzer-only reference: generates identity properties on sample entities. |

## Consumer guide

| Test project needs | Reference |
|---|---|
| Entity types + `EntityOptionsFixture` only (no active-record) | `Forge.Entity.Tests.Fixtures.Core` |
| Active-record methods (`.CreateAsync()`, `.ListAsync()`, etc.) | `Forge.Entity.Tests.Fixtures` |

## File assignment

| File | Description |
|------|-------------|
| `EntityOptionsFixture.cs` | xUnit `IClassFixture` that sets `EntityOptions.BaseIri`. |
| `Sample/Artist.cs` | UuidV5 entity with all scalar CLR types and an inverse collection. |
| `Sample/Album.cs` | Random entity with owning refs to Label, Artist, and Track. |
| `Sample/Label.cs` | PropertyBasedPlain entity that owns a 1:N album collection. |
| `Sample/Track.cs` | Random entity with N:1 to Artist and an inverse ref to Album. |
| `Sample/FeaturedArtist.cs` | Subtype of Artist (ADR-0016) with two additional predicates. |

API Reference

Key interfaces, classes, and extension methods per package — ordered from most fundamental to most derived.

Forge.Execution

Key types

NameDescription
ExecutionCorrelationAmbient correlation record — ExecutionId (GUID) + RequestId. Flows through the whole pipeline.
ExecutionErrorStructured error with Code (string) and Message. Returned by all fail paths.
ExecutionResult<T>Discriminated union: Ok carries T Response; Fail carries ExecutionError.
ExecutionScopeStatic ambient scope; ExecutionScope.Current returns the active ExecutionCorrelation; Use(correlation) creates a scoped context.

Forge.Aspects.Abstractions

Interfaces

NameDescription
IAspectMarker interface for all aspect types (operation, query, message).
IOperationAspectValidates an entity write operation against a SHACL shape.
IMessageAspectValidates a capability command payload against a SHACL shape.
ICapabilityAspectValidates a capability invocation context.
IAspectStoreRegistry for registered aspects, keyed by IRI.
AspectExceptionException thrown when an aspect validation fails; carries SHACL violation details.

Key types

NameDescription
Aspect.NoOpIriSentinel IRI ("urn:forge:aspects:no-op") meaning "do not enforce any aspect".

Forge.Messaging.Abstractions

Interfaces

NameDescription
IMessageProducer<TKey,TValue>Publishes one message to a named topic.
IMessageConsumer<TKey,TValue>Consumes messages from a named topic as an async stream.
IMessageSerializer<T>Serialization seam — converts T to ReadOnlyMemory<byte>.
IMessageDeserializer<T>Deserialization seam — converts ReadOnlyMemory<byte> to T.

Key types

NameDescription
MessageEnvelope<TValue>Broker-agnostic envelope; carries Topic, PartitionKey, Payload, Correlation, TimestampUtc, ContentType, SchemaVersion.

Forge.ObjectStorage.Abstractions

Interfaces

NameDescription
IObjectStoreRead/write blob storage for a single entity type. Provides PutAsync, GetAsync, DeleteAsync.
IObjectStoreProviderResolves a keyed IObjectStore by StoreKey.

Key types

NameDescription
ObjectNotFoundExceptionThrown when a blob cannot be found for the given entity IRI.
[ObjectBearing]Attribute marking an entity class as owning a binary object. Forces MapObjectOperations() to register the eight-route surface.

Forge.Entity

Interfaces

NameDescription
IEntityMarker interface for all entity types. Exposes string Iri and bool IsIdentitySealed.
IEntityOptionsProvides BaseIri and PredicateBaseIri to generators and mappers.
IEntityLoaderLoads a single entity by type and IRI.
ICollectionLoaderLoads a typed collection of entities by IRI list.
IInverseRefLoaderLoads inverse-relation collections (e.g. all Studios managed by an Artist).

Key types

NameDescription
EntityBaseAbstract base implementing IEntity; provides IRI sealing and equality by IRI.
EntityRef<T>Awaitable lazy reference to a related entity. Exposes Iri, IsLoaded, ValueOrThrow, ResolveAsync().
EntityRefCollection<T>Async-enumerable, lazy collection of related entity references. Exposes AddAsync, RemoveAsync, ContainsAsync.
EntitySessionAmbient scope providing the current IEntityLoader to generated code.
EntityOptionsStatic ambient wiring for BaseIri and PredicateBaseIri; override with EntityOptions.Use(options).

Attributes

NameDescription
[Entity]Declares a class as an entity. Parameters: Path (URL segment), PredicatePath (predicate IRI segment).
[Identity]Selects the identity strategy: IdentityGenerator.Random (UUID), IdentityGenerator.PropertyBasedPlain (from [IdentityPart] properties).
[IdentityPart(order)]Marks a property as contributing to the composite IRI suffix.
[Predicate(name)]Declares a scalar or relation property with the given predicate local name.
[Inverse(targetProperty, predicateName)]Declares a read-only inverse-relation collection populated by the inverse loader.
[Enumeration]Marks an entity class as a closed vocabulary of named individuals. Suppresses Create/Update/Delete endpoints.

Forge.Repository

Interfaces

NameDescription
IEntityStoreRead-only entity store; exposes typed load and async-enumerable query.
ITransactionalEntityStoreExtends IEntityStore with transactional write support (used by EntityTransaction).
IEntityRepository<T>Typed helper over IEntityStore; provides FindAsync, LoadAsync, QueryAllAsync.
ISparqlQueryStoreOptional back-end interface exposing raw SPARQL SELECT/ASK queries.
IMultiGraphSparqlStoreExtends ISparqlQueryStore for backends that support multi-graph queries.
IRdfMapper<T>Projects an entity to RDF triples and hydrates it back from an RdfGraph.
IRdfMapperRegistryRegistry of IRdfMapper instances by entity type.

Key types

NameDescription
EntityTransactionUnit of work — fluent Create<T>, Update<T>, Delete, DropGraph. Commit via CommitAsync().
EntityRepository<T>Concrete IEntityRepository<T> wrapping an IEntityStore.
BranchScopeThread-local ambient scope; sets the active branch IRI for all store operations.
ForgeEntityRepositoryBuilderFluent builder returned by AddForgeEntityRepository(). Exposes store key constants and RegisterEntity<T>().
RdfTermValue type representing an IRI, blank node, or literal in the RDF model.
RdfTripleValue type holding (Subject, Predicate, Object) RdfTerms.
EntityAlreadyExistsExceptionThrown when a Create operation targets an IRI that already exists.
EntityNotFoundExceptionThrown by LoadAsync when the entity IRI is not present.

Forge.Operations

Key types

NameDescription

Attributes

NameDescription
[OperationEndpoints]Marks an entity class for REST endpoint registration via MapOperations(). Optional parameter overrides the URL path segment.

Forge.Capability

Interfaces

NameDescription
ICapabilityHandler<TCommand,TResponse>In-process command handler. Implement HandleAsync(command, context, ct) returning ExecutionResult<TResponse>.
ICapabilityDispatcher<TCommand,TResponse>Synchronous in-process dispatcher — routes to the registered handler.

Key types

NameDescription
CapabilityContextExecution context passed to HandleAsync; provides ExecutionCorrelation, AspectIri, and an event accumulator.

Attributes

NameDescription
[Capability]Marks a handler class with its HTTP route (Group, Name, Method) and optionally messaging topic names.

Forge.Capability.Messaging

Interfaces

NameDescription
IAsyncCapabilityDispatcher<TCommand,TResponse>Async dispatcher. PublishAsync (fire-and-forget) and PublishAndWaitAsync (request-reply via PendingReplyRegistry).
ICapabilityMessageConsumer<TCommand,TResponse>SDK-provided consumer; reads one command envelope, delegates to the registered ICapabilityHandler, publishes the reply.

Key types

NameDescription
CapabilityCommandEnvelope<TCommand>Published by the dispatcher; carries Command, Correlation, AspectIri, ReplyToTopic, TimestampUtc.
CapabilityReplyEnvelope<TResponse>Published by the consumer; carries Result, Correlation, TimestampUtc.

Forge.Branch

Key types

NameDescription
BranchEntity type representing a named graph branch; has Name, Description, and DefaultSnapshotIri.
BranchDefaultStatic helper exposing BranchIri — the IRI of the configured default branch (from Forge:Branch:DefaultBranchIri).
BranchProtectionViolationExceptionThrown when an attempt is made to write directly to a snapshot-protected graph.
SnapshotImmutabilityViolationExceptionThrown when a write targets a frozen snapshot graph.

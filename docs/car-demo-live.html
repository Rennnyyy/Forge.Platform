Geometry Assembly

2D top-down diagram assembled from Geometry nodes linked to the active structural tree via GeometryUsage edges. Each edge carries a matrix [a, b, c, d, e, f]; G→G edges compose the matrix of the parent to position sub-geometries (e.g. wheel pairs relative to the car body outline).

3D Geometry Assembly

3D schematic assembled from Geometry3D nodes (OBJ format — plain ASCII, text-editable) and GeometryUsage3D edges, each carrying a 4×4 column-major matrix. One wheel node is reused for all four positions via different placement matrices. Drag to orbit · scroll to zoom.
